stap 1: 1 speler host de server door Operation Flashpoint Server.jar te runnen.
stap 2: Andere spelers (en host) openen Operation Flashpoint met een IDE en runnen main 2 (jar was net te groot voor GITHUB).
stap 3: Voer het IP adres van de host in.
stap 4: Voer een gebruikers naam in.
stap 5:  Begin het spel door een startlocatie in te typen voor de huide spelers. ALleen coordinaten aan de buitenkanten van het bord zijn mogelijk. (zoals 0, 0).
stap 6: 2. Voer de actie uit die u wilt uitvoeren, al is er voor test redenen een een oneindige hoeveelheid aan punten nodig, druk dan op G.
stap 7: Bij het eindigen van uw zet wordt alle spelinformatie gestuurd naar de server, en de zet wordt doorgegeven aan een nieuwe lokale server, opnieuw zal een startlocatie moeten worden gekozen.
stap 8: 4. De meest recente speldata kan worden opgevraagt door op het 'refresh' icoon rechts boven in.
